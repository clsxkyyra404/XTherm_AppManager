XTherm App Manager is the official companion application required to control and manage the [XTherm Module](https://github.com/clsxkyyra404/XTherm). This app provides a clean, modern, and bilingual user interface to interact with your device's root-level thermal mechanisms safely.

> ⚠️ REQUIREMENT: This app strictly requires Root access (Magisk or KernelSU) and the core [XTherm Module](https://github.com/clsxkyyra404/XTherm) flashed on your device to function.

---

## 🚀 Key Features

- SoC Auto-Detection: Automatically detects whether your device is running a Snapdragon or MediaTek chipset and intelligently filters the UI parameters to match your hardware.
- Advanced GPU Sub-Menu: A dedicated, hidden control panel for extreme GPU tuning (Adreno/Mali) without cluttering the main thermal interface.
- Thermal Analytics Dashboard: Real-time monitoring of your active thermal zones, running services, and stopped processes, complete with a manual refresh button.
- One-Click "Kill All": Instantly disable all thermal throttles for your specific chipset safely with a single tap.
- Bilingual UI: Fully supports both English and Indonesian languages, switching dynamically.
- Auto-Update Checker: Never miss a feature drop with built-in update notifications.

---

## 📥 Download & Installation

1. First, make sure you have installed the [XTherm Module](https://github.com/clsxkyyra404/XTherm) via Magisk or KernelSU.
2. Download the latest XTherm-vX.X.apk from the [Releases](../../releases) tab.
3. Install the APK normally.
4. Open the app and grant Root Access when prompted.
5. You're ready to break the thermal limits!
